# Sistema de Busca Otimizado para Catálogo de Produtos - MegaStore

## Descrição

Este projeto implementa um sistema de busca otimizado para o catálogo de produtos da MegaStore utilizando a linguagem Rust. O objetivo é melhorar a eficiência da busca em um catálogo com milhões de produtos.

O sistema utiliza estruturas de dados como HashMap para indexação de produtos e grafos para recomendação de produtos relacionados.

---

## Tecnologias Utilizadas

- Rust
- Cargo
- HashMap
- Estrutura de Grafos
- Testes com cargo test
